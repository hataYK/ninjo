# プロダクト仕様書 - Ninjo

## 概要

Ninjo はドット絵の「分身（アバター）」を勉強で育てる学習アプリ。
書籍のページ数ベースで学習計画を立て、タスクをこなしてメモを書くと、
AIがメモからスキルを抽出してアバターに蓄積する。

計画を立てるのも自分、実行するのも自分、育つのも自分。
この自己完結のループをドット絵のゲーム画面として可視化することで、
地味な勉強が「自分を育てる体験」に変わる。

### コアループ

```
計画作成 → タスク完了 → メモ記入 → AIスキル抽出 → アバター成長
```

## ターゲットユーザー

- **ペルソナ**: 大学生〜若手社会人。資格取得・スキルアップを計画的に進めたい人
- **課題**: 計画は立てるけど続かない。成長の実感が湧かず、モチベーションが持続しない
- **求めているもの**: 自分の成長を具体的なスキルとして可視化し、「ちゃんと前に進んでる」と実感できる体験

## コア機能

### F0. ユーザー認証

| 項目 | 内容 |
|---|---|
| 概要 | メール + パスワードで認証 |
| 方式 | JWT（httpOnly Cookie）|

### F1. 計画作成 + AIレビュー

| 項目 | 内容 |
|---|---|
| 概要 | 教材名・総ページ数・期限を入力して学習計画を作成する |
| AIの役割 | 可処分時間と（あれば）過去実績から、1日あたり必要ページ数を算出し、妥当性をレビュー |
| 入力 | 教材名、総ページ数、目標期限 |
| 出力 | AIレビュー結果（1日の必要ページ数、現実的かの判断、アドバイス） |

### F2. デイリータスク

| 項目 | 内容 |
|---|---|
| 概要 | 全計画を横断して「今日やること」を1画面に表示 |
| AIの役割 | 計画から今日分のタスクを自動提案 |
| ユーザー操作 | AIの提案をそのまま使う or 自分で調整 |

### F3. タスク完了 + メモ

| 項目 | 内容 |
|---|---|
| 概要 | チェックボックスで完了。完了時に学んだ内容をメモする |
| メモ | 完了タイミングで入力。自由テキスト |
| データ活用 | メモは F9（スキル抽出）と F5（学習サマリー）の素材になる |

### F4. 進捗分析 + 軌道修正

| 項目 | 内容 |
|---|---|
| 概要 | 過去実績からペースを分析し、遅れを検知してリスケ提案 |
| AIの役割 | 実績が溜まったら「1時間あたりXページ」を推定。遅れ時にリスケ案を生成 |
| 表示 | 計画ごとの進捗率、予定 vs 実績の差分 |

### F5. 学習サマリー

| 項目 | 内容 |
|---|---|
| 概要 | 本が完了した時、全メモからAIが学習サマリーを自動生成 |
| 出力 | 習得した知識、成長ポイント、復習推奨箇所 |
| トリガー | 全ページ完了時に自動生成 |

### F6. 可処分時間設定

| 項目 | 内容 |
|---|---|
| 概要 | 曜日ごとに勉強できる時間を設定 |
| 入力 | 日〜土それぞれの時間（h単位、Go time.Weekday 準拠） |
| 用途 | AIの計画レビュー・デイリータスク配分の基礎データ |

### F7. アバター管理

| 項目 | 内容 |
|---|---|
| 概要 | ドット絵プリセットから自分の分身を選択 |
| 表示 | ホーム画面の中央にアバターを配置。スキル一覧を併せて表示 |
| カスタマイズ | プリセット（8-12種）から選択。着せ替え・アニメーションはv1.1以降 |

### F8. ルーム / WiFi連携

| 項目 | 内容 |
|---|---|
| 概要 | WiFi SSIDと場所を紐付け、アバターの背景が自動で切り替わる |
| 場所例 | 自宅の部屋、研究室、カフェ、図書館 |
| 制約 | ブラウザではSSID取得不可。ネイティブアプリ化時に本格対応 |
| MVPスコープ | v1.1。MVPではデフォルト背景のみ |

### F9. スキル抽出・蓄積

| 項目 | 内容 |
|---|---|
| 概要 | タスク完了メモからAIがスキルを抽出し、アバターに蓄積 |
| AIの役割 | メモを解析し、具体的なスキル（例: "AWS: VPC設計"）を抽出 |
| ユーザー操作 | AI提案を確認・編集 + カテゴリ選択（ハイブリッド方式） |
| カテゴリ | プログラミング、インフラ、ネットワーク、セキュリティ、データベース 等 |

## MVP スコープ（v1.0）

**含む:**
- F0: ユーザー認証
- F6: 可処分時間設定
- F7: アバター管理（ドット絵プリセット選択 + スキル表示）
- F1: 計画作成 + AIレビュー
- F2: デイリータスク
- F3: タスク完了 + メモ
- F9: スキル抽出・蓄積

**v1.1 以降:**
- F8: ルーム / WiFi連携
- F4: 進捗分析 + 軌道修正
- F5: 学習サマリー
- アバターの着せ替え・アニメーション
- 通知・リマインダー

## 画面構成（MVP）

```
[ログイン / サインアップ]
        ↓
[ホーム]  ← アバターが中央。左右にタスク・計画を表示
        ↓
[計画一覧]       ← 登録中の学習計画 + 各進捗率
        ↓
[計画作成]       ← 教材・ページ数・期限入力 → AIレビュー → 確定
        ↓
[アバター詳細]   ← スキル一覧・カテゴリ別表示
        ↓
[設定]           ← 可処分時間 + アバター変更
```

## 成功指標

| 指標 | 目標 |
|---|---|
| 計画完了率 | 作成した計画の50%以上が完了 |
| デイリータスク消化率 | 平均60%以上 |
| メモ記入率 | タスク完了時の70%以上でメモあり |
| スキル蓄積数 | 計画完了あたり平均5個以上 |
| メモ→スキル変換率 | メモの80%以上から1つ以上のスキル抽出 |
| 継続率 | 7日後リテンション40% |
